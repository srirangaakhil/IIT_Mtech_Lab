Instructions to Run "q1":
1. To compile the program execute the following command:
	
	javac -cp .:./jar_files/* q1.java

2. To execute one of the files use the following command:

	java -cp .:./jar_files/* q1 <input file>

example: java -cp .:./jar_files/* q1 ./q1_sample_files/sample.pdf

The output file is created in the current directory with name doc.pdf

The following jar files need to be present:
commons-collections4-4.4.jar
commons-compress-1.18.jar
dom4j-1.6.1.jar
pdfbox-app-2.0.17.jar
poi-4.1.0.jar
poi-ooxml-4.1.0.jar
poi-ooxml-schemas-4.1.0.jar
poi-scratchpad-4.1.0.jar
xmlbeans-3.1.0.jar

Some sample files can be found in "q1_sample_files" directory.
